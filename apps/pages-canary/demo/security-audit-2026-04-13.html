Open Findings

High No Automated Secret Rotation

Kubernetes Secrets and Cloudflare API tokens are manually created and never rotated. A leaked token remains valid indefinitely.

Fix: Implement cert-manager-based secret rotation with 90-day expiry. Add ExternalSecrets operator for Cloudflare tokens.
Medium Self-Signed Registry Certificate

Container Registry uses a self-signed CA certificate. Docker pulls require --tls-verify=false which disables certificate validation.

Fix: Add internal CA to node trust stores or use cert-manager with internal issuer for registry certificates.
Low Ceph Dashboard Without Authentication

Storage dashboard is accessible on the internal network without login. While not externally exposed, lateral movement is possible.

Fix: Enable Ceph dashboard SSO via Cloudflare Access or add basic-auth ingress annotation.
