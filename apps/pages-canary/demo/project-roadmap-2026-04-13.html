CANARY

Project Roadmap

Quarterly milestones and priorities for Second Brain

Q1 2026 Current
Responsive UI Overhaul Completed

Full mobile support with touch-optimized navigation, responsive grids, and safe-area insets for iOS.

Cloudflare Access Integration Completed

Per-user vault scoping via Cloudflare Access OAuth. Bearer tokens auto-provisioned from Access identity.

GitOps Deployment Pipeline In Progress

Automated canary deployment from isolated worktrees with ArgoCD auto-sync and Discord feedback loops.

E2E Testing Suite In Progress

Playwright-based end-to-end tests covering hash routing, authentication flow, and API interactions.

Q2 2026 Upcoming
Multi-Model Chat Switching Planned

Switch between Ollama, OpenAI, and Anthropic models mid-conversation with context preservation.

Vault Sharing & Permissions Planned

Share vaults between users with read-only and read-write permission levels. Access grants via Cloudflare Access groups.

Webhook Integrations Planned

Outgoing webhooks on vault events (commit, chat, merge). Incoming webhooks for CI/CD status updates.

Q3 2026 Future
Plugin System Planned

Extensible plugin architecture for custom vault processors, AI providers, and chat commands.

Real-time Collaboration Planned

Multi-user editing with conflict-free replicated data types (CRDTs) for concurrent vault modifications.
