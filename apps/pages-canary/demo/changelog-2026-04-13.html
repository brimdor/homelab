Release Changelog

Version history for the PageVault platform

v2.4.0 April 10, 2026
  • Feature SSE streaming for chat API — tokens delivered in real-time instead of batch response
  • Feature Vault-level model selection — choose between Ollama, OpenAI, and Anthropic per vault
  • Fix Chat context window correctly respects 128k token limit instead of truncating at 32k
  • Fix Git sync no longer fails on repositories with LFS-tracked files
  • Feature Added /model slash command in chat to switch providers mid-conversation
v2.3.1 March 28, 2026
  • Fix Resolved race condition in bi-directional git sync that caused duplicate commits on fast successive saves
  • Fix GitOps controller sync no longer creates duplicate Ingress records when canary and prod share same image tag
  • Deprecate The /api/v1/chat endpoint is deprecated in favor of /api/v1/vaults/:id/chat (removal in v3.0)
v2.3.0 March 15, 2026
  • Feature Multi-vault dashboard with aggregated health metrics
  • Feature SSO Provider integration for per-user vault scoping
  • Breaking Bearer token format changed from JWT HS256 to RS256 — existing tokens must be re-issued
  • Feature Added Prometheus metrics endpoint at /metrics for operator observability
  • Fix Container image now correctly respects TZ environment variable for log timestamps
v2.2.0 February 22, 2026
  • Feature Local-first AI inference via Ollama with automatic cloud fallback
  • Feature Responsive UI overhaul — full mobile support with touch-optimized navigation
  • Fix PVC multi-attach errors resolved by switching from ReadWriteMany to ReadWriteOnce with Distributed Storage
